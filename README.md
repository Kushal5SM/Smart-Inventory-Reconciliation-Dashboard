# Smart Inventory Reconciliation Dashboard (SIRD)

## Overview
The Smart Inventory Reconciliation Dashboard (SIRD) is a Python and Excel based inventory reconciliation solution developed during my internship at Blue Enertech Pvt. Ltd. (NPOWER Batteries).

The system automates comparison between ERP inventory records and physical stock counts, reducing reconciliation time from 2.5 days to approximately 90 minutes.

## Key Features

- Automated inventory reconciliation
- ERP vs Physical stock comparison
- Variance computation
- Severity-based discrepancy tagging
- Historical discrepancy tracking
- Excel dashboard generation
- Inventory health reporting

## Technology Stack

- Python
- Pandas
- NumPy
- OpenPyXL
- XlsxWriter
- SQLite
- Microsoft Excel
- Matplotlib

## Business Impact

- 93% reduction in reconciliation time
- Zero transcription errors
- Detection of ₹48,000+ stock discrepancies
- Historical inventory trend analysis

## Dashboard Workflow

1. Import ERP inventory data
2. Import physical stock count data
3. Standardize and clean data
4. Calculate variances
5. Generate severity tags
6. Create management dashboard
7. Export reports
